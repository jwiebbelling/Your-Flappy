# Your Flappy

Your Flappy is a lightweight, browser-based Flappy Bird-style game with support for custom character uploads.

##  Play Online

Play in your browser: [https://jwiebbelling.github.io/Your-Flappy/](https://jwiebbelling.github.io/Your-Flappy/)

## Features

- Classic tap-to-fly gameplay
- Browser-only (no build step required)
- Custom character upload support
- Sprite and audio assets included

## Controls

- Spacebar / click / tap: flap
- Restart when game over: use the on-screen prompt

## Project Files

- `index.html` - main page and overlays
- `game.js` - game loop and state
- `physics.js` - movement and collision logic
- `renderer.js` - drawing and visual updates
- `input.js` - keyboard/mouse/touch input
- `uploader.js` - custom character upload handling
- `constants.js` - shared game constants
